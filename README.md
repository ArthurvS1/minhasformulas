# Minhas Fórmulas

Projeto integrador para a disciplina de Programação com Acesso a Banco de Dados, envolvendo as matérias de Física, Química e Matemática. Para esse projeto foram usadas as seguintes bibliotecas/frameworks:

- [Bootstrap](https://getbootstrap.com/)
- [Express](http://expressjs.com/pt-br/)
- [jQuery](http://code.jquery.com/)
- [FontAwesome](https://fontawesome.com/)
- [KaTeX](https://khan.github.io/KaTeX/)
- [sqlite3](https://www.npmjs.com/package/sqlite3)
