# Minhas Fórmulas

Projeto integrador para a disciplina de Programação com acesso a banco de dados, envolvendo as matérias de Física, Matemática e Química. Para esse projeto foram usadas as seguintes bibliotecas/frameworks:

- [Bootstrap](https://getbootstrap.com/)
- [Express](http://expressjs.com/pt-br/)
- [jQuery](http://code.jquery.com/)
- [FontAwesome](https://fontawesome.com/)
- [KaTeX](https://khan.github.io/KaTeX/)
- [sqlite3](https://www.npmjs.com/package/sqlite3)
